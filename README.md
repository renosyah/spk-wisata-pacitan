# Project SPK wisata pacitan

## Kebutuhan

- PHP 7
- MysqlDB or MariaDB

## Cara Menjalankan

- buka cmd di direktori ini lalu ketik
    ```
        
        php -S {YOUR_IP}:80

    ```
    atau

     ```
        
        php -S localhost:80

    ```
- buka web browser dan menuju url `localhost:80`
